# Lesson

This lesson covers the basic commands  `date`, `echo`, `man` and `clear`. It also presents the main editing shortcuts, the shell history and **tab** completion.

# Setup

# Tasks

1. Use the command `date`. Then use the command `date +%Y-%m-%d`. Finally, take a look at the options with `date --help`.

2. Use the command `echo`. Then use the command `echo test`. Finally, try the command `echo test > file`. What happened?

3. Use the command `man`. Then use the command `man echo` (use the **q** key to quit). Finally, try `man date`.

4. Use the **up** key to go back in history and the **enter** key to execute the current command. Test also the **down** key.

5. Test other keys like **left**, **right**, **backspace**.

6. Use the command `clear`. Then repeat a previous command. Now try the **control+l**` shortcut.

7. Now just type `man` but do not press **enter**. Instead hit the **tab** key twice. What happened?

# Check

2. Verify that the file `file` exists and contains `test`.
